## Rodando localmente

Clone o projeto

```bash
  git clone https://github.com/gabrielbarros23/SpotifyClone-Tailwind-Next
```

Entre no diretório do projeto

```bash
  cd SpotifyClone-Tailwind-Next
```

Instale as dependências

```bash
  npm install
```

Inicie o servidor

```bash
  npm run dev
```
